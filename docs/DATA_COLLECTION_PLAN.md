### БЛОК E: Заклинания — списки (КРИТИЧЕСКИЙ)
**Для шага 4.4.6**

Нужны заклинания **1-го уровня** для виззарда (персонаж всегда создаётся на 1 уровне).

#### Формат данных
```js
{
  name: 'Волшебная стрела',
  level: 1,
  school: 'Воплощение',
  classes: ['Волшебник'],
  isCantrip: false,
  description: 'Создаёт три светящихся снаряда, каждый наносит 1к4+1 силовым уроном.',
  ritual: false,
  concentration: false,
  source: 'PHB'
}
```

#### Что нужно собрать:
- Кантрипы: Бард (2), Жрец (3), Друид (2), Волшебник (3), Колдун (2), Чародей (4), Изобретатель (2)
- Заклинания 1 уровня: для всех магических классов

Все E1–E16 закрыты единым файлом `js/data/spells.js` — данные извлечены из `window.LIST` на dnd.su/spells/.

- [x] **E1–E7** — Кантрипы всех 7 кастующих классов (44 кантрипа)

## 4. Порядок сбора (рекомендованный)

Порядок отражает приоритет и зависимости:

1. **БЛОК A** — Классы (описания, роли, фишки) — основа большинства выборов
2. **БЛОК B** — Подклассы Жреца/Колдуна/Чародея — нужны для кнопки «Далее» на шаге Класса
3. **БЛОК C** — Расы (описания, умения, подрасы)
4. **БЛОК D** — Предыстории (описания)
5. **БЛОК E** — Заклинания — самый трудоёмкий, начать параллельно с остальными
6. **БЛОК B (остаток)** — Подклассы прочих классов (не блокируют прогресс)
7. **БЛОК F** — Дополнения (можно в последнюю очередь, шаг не блокирует виззард)

## 6. Формат итоговых файлов данных

По завершении сбора данные пойдут в новые файлы:

| Файл | Содержимое |
|---|---|
| `js/data/class_descriptions.js` | Описания, роли, фишки, RP-сложность классов |
| `js/data/subclass_descriptions.js` | Флаворные описания подклассов |
| `js/data/race_descriptions.js` | Описания рас, умений, подрас |
| `js/data/background_descriptions.js` | Описания предысторий |
| `js/data/spells.js` | Списки кантрипов и заклинаний 1 ур. по классам |
| `js/data/sources.js` | Данные дополнений (категории, описания, совместимость) |
