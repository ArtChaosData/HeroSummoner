# Skill: ui-builder
# Goal: Генерация динамического UI на основе JSON-данных.

## 💎 Принципы Senior-фронтенда
1. **Data over Code**: Названия классов, рас и умений **запрещено** хардкодить в JS. Читай их из JSON через `@js/data/`.
2. **Responsive Tiers**: 
   - Mobile: 1 колонна, карточки-аккордеоны.
   - Desktop: До 3 колонн, развернутые описания.
3. **BEM & Variables**: Используй CSS-переменные для темизации и БЭМ для классов.

## 🧪 Визуальная верификация
После внесения правок в UI:
1. **Staging**: Запусти сервер `python -m http.server 3000` [28, 29].
2. **Verification**: Если доступно расширение Claude в Chrome, требуй сравнения со скриншотом дизайна.
3. **Checklist**: Убедись, что переключатель редакций (5e14/5e24) корректно фильтрует данные на экране.

## ⚠️ Обязательное условие
Перед кодированием компонента **всегда** читай соответствующий раздел в `docs/SPECIFICATION.md`.
